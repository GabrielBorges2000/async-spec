# Async-spect

App para ler o async storage de aplicações React Native que irá receber os dados atrávez de um web hook local.

## Recommended IDE Setup

- [VSCode](https://code.visualstudio.com/)

## Project Setup

### Install

```bash
$ pnpm install
```

### Development

```bash
$ pnpm dev
```

### Build

```bash
# For windows
$ pnpm build:win

# For macOS
$ pnpm build:mac

# For Linux
$ pnpm build:linux
```
